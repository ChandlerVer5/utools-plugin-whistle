# whistle

> 提供给 uTools 的 Whistle 插件，方便操作 Whistle 规则
> 更改自： [utools-whistle](https://github.com/MrLeo/utools-whistle)


# Features

- [x] 为手机 SS 代理软件生成 Whistle 二维码
- [x] Whistle 规则开关
- [x] 检查并初始化 Whistle 环境
- [x] 获取 Whistle 状态
- [x] 启动/关闭/刷新 Whistle
- [x] 启动后，会自动设置系统代理（chandlerVer5）
- [ ] 启动后，是否需要多地址切换，显示 二维码（chandlerVer5）
